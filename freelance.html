Photography Portfolio
Technologies/Libraries Used: HTML / CSS / JS / jquery / LazyLoad.js

A photography & tattoo portfolio website to showcase work. Utilizes CSS Flexbox to align images in a single side-scrolling column. Images are lazy loaded using a library that leverages the Intersection Observer API and callback functions that execute whenever an image enters the viewport. Images fade in as the user scrolls along.
